# 个人产品导航首页 - 项目开发说明书

## 1. 项目概述

### 1.1 项目背景
- **项目名称**：heshanws-homepage
- **项目类型**：React 单页应用（SPA）
- **核心目标**：作为个人产品矩阵的统一入口页面，帮助用户快速了解和跳转至各个产品
- **目标用户**：所有访客（潜在客户、现有用户、技术同行）

### 1.2 产品列表
| 产品名称 | 子域名 | 简介 |
|---------|--------|------|
| Highland Game | highlandgame.heshanws.top | 高地游戏 |
| Portfolio Analyzer | portfolioanalyzer.heshanws.top | 投资组合分析工具 |
| QuickMan | quickman.heshanws.top | 效率工具 |
| QuickShare | quickshare.heshanws.top | 分享工具 |

### 1.3 主网站
- **URL**：https://www.heshanws.top/

---

## 2. 功能需求规格

### 2.1 功能清单

| 功能 | 优先级 | 描述 |
|-----|--------|------|
| 产品卡片展示 | P0 | 展示所有产品，带名称、简介、图标 |
| 外部链接跳转 | P0 | 点击卡片跳转至对应子域名 |
| 响应式布局 | P0 | 适配桌面端和移动端 |
| 简洁导航 | P1 | 顶部导航栏，包含主站标识 |
| 社交链接 | P2 | 底部展示社交媒体链接（GitHub、邮箱等） |

### 2.2 用户流程
```
用户访问主页 
    ↓
浏览产品卡片列表 
    ↓
点击感兴趣的产品卡片 
    ↓
新窗口打开对应产品页面
```

### 2.3 页面结构
- **Header**：网站标题/Logo
- **Main Content**：产品卡片网格
- **Footer**：版权信息、社交链接

---

## 3. 技术架构设计

### 3.1 技术栈选型

| 技术 | 选择 | 理由 |
|------|------|------|
| 框架 | React 18 | 成熟稳定，生态完善 |
| 构建工具 | Vite | 快速、轻量、开发体验好 |
| 语言 | TypeScript | 类型安全，减少错误 |
| 样式 | Tailwind CSS | 原子化CSS，开发效率高 |
| 部署 | 静态资源 | 可部署至任意Web服务器 |

### 3.2 项目结构
```
heshanws-homepage/
├── public/
│   └── favicon.ico
├── src/
│   ├── assets/                 # 静态资源
```

### 3.3 数据模型
```typescript
interface Product {
  id: string;
  name: string;
  subdomain: string;        // e.g., "highlandgame"
  url: string;              // e.g., "https://highlandgame.heshanws.top"
  description: string;
  icon: string;             // 图标路径或emoji
  accentColor: string;      // 卡片强调色
}
```

## 4. UI/UX 设计规范

### 4.1 设计系统

**色彩方案**
| 用途 | 色值 |
|------|------|
| 背景色 | #FFFFFF |
| 文字主色 | #1F2937 |
| 文字次色 | #6B7280 |
| 强调色 | #3B82F6 |
| 卡片背景 | #F9FAFB |
| 边框色 | #E5E7EB |

**字体**
- 主字体：Inter, system-ui, sans-serif
- 标题：Font-weight 600-700
- 正文：Font-weight 400

**间距系统**
- 基础单位：4px
- 卡片间距：24px
- 容器内边距：16px (mobile) / 32px (desktop)

### 4.2 布局规范

**桌面端 (≥768px)**
- 最大宽度：1024px
- 产品卡片：2列网格
- 卡片尺寸：宽度自适应，高度约200px

**移动端 (<768px)**
- 产品卡片：单列堆叠
- 全宽布局

### 4.3 组件规格

**ProductCard 组件**
- 尺寸：自适应容器，min-height 180px
- 圆角：12px
- 阴影：sm (0 1px 2px rgba(0,0,0,0.05))
- 悬停效果：阴影增强 + translateY(-2px)
- 图标大小：48px
- 标题字号：18px / font-weight 600
- 描述字号：14px / color gray-500

**Header 组件**
- 高度：64px
- 内容：网站标题居中
- 背景：透明或微透明

**Footer 组件**
- 内容：版权信息 + 年份
- 社交图标链接
